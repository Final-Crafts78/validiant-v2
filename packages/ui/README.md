# UI Package

Shared UI components library for web and mobile applications.

## Components

### Atoms
- Button
- Input
- Text
- Icon
- Badge
- Avatar

### Molecules
- FormField
- Card
- Modal
- Dropdown
- DatePicker

### Organisms
- Navigation
- Header
- Sidebar
- TaskCard
- ProjectCard

## Usage

```typescript
import { Button, Card, Modal } from '@validiant/ui';
```

## Design System

- Consistent styling across platforms
- Theme support (light/dark)
- Accessibility built-in
- Responsive design
