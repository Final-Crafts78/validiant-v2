# Shared Package

Shared types, utilities, constants, and schemas used across all applications.

## Contents

### Types
- User types
- Task types
- Project types
- API response types
- Common interfaces

### Utils
- Date formatting
- Validation helpers
- String utilities
- Number utilities
- Array utilities

### Constants
- API endpoints
- Status codes
- Error messages
- Configuration values

### Schemas
- Zod validation schemas
- Shared across frontend and backend
- Type-safe validation

## Usage

```typescript
import { User, formatDate, API_ENDPOINTS } from '@validiant/shared';
```
